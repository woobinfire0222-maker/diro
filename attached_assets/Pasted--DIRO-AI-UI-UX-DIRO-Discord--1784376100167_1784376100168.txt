# DIRO (디로) - AI 웹사이트 개발 프롬프트

당신은 세계 최고 수준의 풀스택 개발자이자 UI/UX 디자이너입니다.

"DIRO(디로)"라는 Discord 서버 맞춤 제작 플랫폼을 처음부터 끝까지 제작하세요.

절대 데모 수준으로 만들지 말고 실제 서비스처럼 구현하세요.

==================================================
프로젝트 소개
==================================================

DIRO는 Discord 서버를 돈을 받고 맞춤 제작해주는 플랫폼입니다.

사용자는 Discord로 로그인하여 원하는 서버를 신청하고,
상담사와 실시간 채팅을 통해 서버를 제작합니다.

상담사는 사이트 내부에서 Discord와 거의 동일한 UI를 사용하여 서버를 제작하고,
완성 후 Discord Bot을 이용하여 실제 Discord 서버에 적용합니다.

전체적인 퀄리티는

- Toss
- Discord
- Notion
- Linear
- Vercel Dashboard

수준을 목표로 합니다.

==================================================
미리보기
==================================================

상담사는

"미리보기 전송"

버튼을 누르면

현재 서버 상태를

신청자에게 전송합니다.

신청자는

Discord와 동일한 형태의 미리보기 UI를 확인할 수 있습니다.

==================================================
송금 요청
==================================================

상담사는

"송금 요청"

버튼을 누릅니다.

가격 입력창

예)

5000

10000

35000

입력 완료 시

자동으로

다음 딥링크를 생성합니다.

supertoss://send?bank=토스&accountNo=190839534245&amount=[가격]

[가격]에는 상담사가 입력한 숫자가 들어갑니다.

신청자의 채팅창에는

"송금하기"

버튼이 생성됩니다.

버튼 클릭 시

위 딥링크를 실행합니다.

==================================================
기타
==================================================

모든 화면은 실제 서비스 수준으로 제작합니다.

빈 화면이나 Placeholder를 최소화합니다.

필요한 텍스트와 안내문은 자연스럽게 작성합니다.

로딩 화면, Skeleton UI, Toast 알림, 모달, 드롭다운, 확인창 등을 구현합니다.

오류 처리와 예외 처리도 포함합니다.

Dark Mode와 Light Mode를 지원합니다.

사이트 전체가 하나의 완성된 SaaS 서비스처럼 보이도록 구현합니다.

코드는 유지보수가 쉽도록 컴포넌트 단위로 분리하고, 폴더 구조도 체계적으로 구성합니다.
